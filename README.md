# 📚 EpiStudy AI

> Learn Smarter.

EpiStudy AI is a full-stack AI-powered study assistant designed to help students generate summaries, flashcards, study plans, explanations, and practice tests instantly using modern LLMs.

Built using React, Flask, Groq AI, and LangChain.

---

# 🚀 Live Demo

Frontend: [https://epistudy.vercel.app] (https://epistudy.vercel.app)

Backend: [https://epistudy.onrender.com] (https://epistudy.onrender.com)

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/ssr1807/EpiStudy.git
cd EpiStudy
```

---

# 🔹 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# 🔹 Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

# 🔑 Environment Variables

Create a `.env` file inside backend folder:

```env
GROQ_API_KEY=your_api_key_here
```
